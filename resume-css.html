CSS Fundamentals Guide

1. Linking CSS to HTML

To link an external CSS file to your HTML:

        <link href="style.css" rel="stylesheet">

2. Writing CSS

CSS can be written in two main ways:

  • External CSS: Link it using the example above.
  • Inline <style> Tag: 
                    <style>
                    p { color: blue; }
                </style>

3. CSS Selectors

  • Type selectors: p, h1
  • Class selector: 
                    .example { color: red; }
  • ID selector: 
                    #unique { font-size: 20px; }
  • Attribute selector: 
                    input[type="text"] { border: 1px solid black; }
  • Pseudo-classes: 
                    a:hover { text-decoration: underline; }

6. Flexbox in Detail

Flexbox is a layout model that allows elements to align and distribute space effectively within a container:

Basic Flexbox Properties:

  • display: Defines a flex container. 
    display: flex;
  • flex-direction: Specifies the direction of the flex items. 
    row | row-reverse | column | column-reverse
  • justify-content: Aligns items horizontally. 
    flex-start | center | flex-end | space-around | space-between | space-evenly
  • align-items: Aligns items vertically. 
    flex-start | center | flex-end | stretch
  • align-content: Aligns multiple rows. 
    flex-start | center | flex-end | space-around

7. Media Queries in Detail

Media queries help create responsive designs:

  • Basic syntax: 
                    @media (condition) {
                    /* Styles */
                }
  • Common conditions: 
                    max-width: 600px; /* For smaller screens */
                min-width: 768px; /* For tablets and up */
                orientation: landscape; /* When the device is in landscape mode */

Example:

        @media (max-width: 768px) {
            body {
                font-size: 14px;
            }
        }

Combined queries:

        @media (min-width: 600px) and (max-width: 900px) {
            .box-model {
                background-color: pink;
            }
        }
